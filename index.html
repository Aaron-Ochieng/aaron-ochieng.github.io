Flask Framework

I chose to use Flask for my GitHub portfolio for several reasons: Flask is a microframework that makes it simple and straightforward to set up a web application. This simplicity allows me to focus on the logic of my application rather than the intricacies of the framework itself. Unlike other frameworks that come with pre-existing third-party libraries, Flask allows me to choose my own tools and libraries. This gives me the flexibility to pick the best tools for my specific needs. Despite its simplicity, Flask is capable of scaling up to complex applications. This means that as my portfolio grows and becomes more complex, Flask will be able to handle it. Flask has a large and active community. This means that if I run into any issues or need help, there are plenty of resources available. Working with Flask also provides a great learning experience. It allows me to gain a deeper understanding of web development and Python.
